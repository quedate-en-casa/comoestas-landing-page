Welcome to Smooth Landing page

Smooth is a HTML5 Landing Page Build using Gulp, SCSS, and Bootstrap 4. It was designed to promote your App, services or business projects.

This template fully responsive, easy to develop, and look great in the high resolution screen

Smooth can be used for all kind of startup or company like: software, (SaaS) service, app, platforms and many more. Don’t wait – launch your startup now!

Initialization

Start development using gulp

Steps 1
Install node.js

Steps 2
Run in the terminal npm install gulp-cli -g

Steps 3
Run in the terminal npm install gulp -D
More info for isntalazion Gulp pleas click here

Steps 3
run npm install

Steps 4
Start develop your template run gulp dev

Image Retina

Smooth is Retina ready, look great in the high resolution screens

for image optimization please provide 2 images, by adding @ 2x to the second image which is twice the size of the first image for high resolution screens

Example :

Default image :

Image name : using-software.png

High resolution image :

Image name : using-software@2x.png

HTML Code : <img src="assets/images/using-software.png" alt=""/>
